# 公司深度分析全框架（全行业适配版）

## 文档说明

本框架适配持有公司财报、财务报表数据的全维度深度分析需求，核心逻辑为「财报真实性校验→三维度财务深度拆解→分行业差异化分析→同业对标与护城河验证→风险识别与压力测试→估值与价值判断→动态跟踪体系搭建」，可直接用于投资分析、信贷尽调、企业内部经营分析、并购标的研判等场景，可根据细分行业灵活调整核心权重与分析重点。

***

## 第一部分 分析前置准备与财报真实性校验（所有分析的前提，必做）

### 第一部分全行业适配总规则

1. 本部分4个子模块按「商业模式→行业属性→竞争格局→分析目标」的逻辑递进，前序模块结论为后序模块提供支撑，后序模块结论反向验证前序模块；
2. 不同行业需按权重调整分析侧重点，核心权重规则：
   - 周期/制造类：1.1商业模式（含话语权）> 1.3竞争格局 > 1.2行业属性 > 1.4分析目标
   - 消费类：1.1商业模式 > 1.2行业属性 > 1.3竞争格局 > 1.4分析目标
   - 监管类（医药/公用/金融）：1.2行业属性 > 1.1商业模式 > 1.3竞争格局 > 1.4分析目标
3. 所有数据需满足「校验规则」，异常值需标注原因，避免基础数据错误；
4. 状态标注统一规范：✅ 已有 / ⚠️ 待提取 / 🔍 待搜索 / ⚙️ 待计算。

#### 2.2 会计政策与估计一致性核查

##### 模块目标

核查公司会计政策、会计估计的**连续性、合规性、合理性**，量化政策/估计变更、会计差错更正对利润、资产的影响幅度，判断是否存在盈余管理/利润操纵空间，同时验证财报数据的纵向（历史）、横向（行业）可比性，为后续财务分析奠定统一、可比的口径基础。

## 第二部分 核心财务三维度深度分析

### 第二部分全行业适配总规则

1. 本部分4个子模块按「资产负债表→利润表→现金流量表→杜邦分析」的逻辑递进，前序模块结论为后序模块提供支撑，后序模块结论反向验证前序模块；
2. 不同行业需按权重调整分析侧重点，核心权重规则：
   - 重资产/周期类：模块1资产负债表（含资产质量、偿债能力）> 模块2利润表 > 模块3现金流量表 > 模块4杜邦分析
   - 轻资产/成长类：模块2利润表 > 模块3现金流量表 > 模块1资产负债表 > 模块4杜邦分析
   - 金融类：模块1资产负债表（含资本充足率）> 模块3现金流量表 > 模块2利润表 > 模块4杜邦分析
3. 所有数据需满足「校验规则」，异常值需标注原因，避免基础数据错误；
4. 状态标注统一规范：✅ 已有 / ⚠️ 待提取 / 🔍 待搜索 / ⚙️ 待计算。

### 模块1 资产负债表分析

#### 模块目标

围绕**资产质量、偿债能力、资本结构、经营扩张合理性**四大核心维度，完成以下分析目标：

1. 穿透判断公司资产的**真实性、变现能力、盈利贡献、减值风险**，识别低效/无效资产；
2. 量化评估公司**短期/长期偿债能力**，判断债务风险与安全边际；
3. 分析公司资本结构的合理性，验证产业链话语权、融资成本与股东回报能力；
4. 判断产能扩张、资本开支的节奏与合理性，验证公司经营战略与业绩增长的匹配性；
5. 与第一部分财报真实性校验结果交叉验证，识别财务异常风险。

### 模块2 利润表分析

#### 模块目标

围绕**成长性、盈利能力、成本费用管控、盈利质量**四大核心维度，完成以下分析目标：

1. 拆解营收与利润的增长来源，区分**量增/价涨/结构优化**驱动，判断增长的可持续性；
2. 分析公司定价能力、盈利水平，通过行业对标验证成本优势与盈利竞争力；
3. 穿透成本结构与费用管控效率，判断公司降本增效能力与经营管理水平；
4. 严格区分**持续性核心盈利与一次性非经常性损益**，识别利润操纵风险，验证盈利的真实性与含金量；
5. 与第一部分财报真实性校验结果交叉验证，识别财务异常风险，联动资产负债表、现金流量表形成分析闭环。

### 模块3 现金流量表分析

#### 模块目标

围绕**经营造血能力、盈利真实性、投资扩张效率、筹资偿债安全、自由现金流价值**五大核心维度，完成以下分析目标：

1. 交叉验证利润表盈利的真实性与含金量，识别收入/利润虚增风险，完成财报真实性校验的闭环验证；
2. 拆解经营现金流结构与趋势，判断公司主营业务的持续造血能力、回款能力与经营稳定性；
3. 穿透投资现金流明细，判断产能扩张节奏、资本开支效率、战略布局的合理性，验证公司长期增长逻辑；
4. 分析筹资现金流结构，判断公司融资能力、债务期限匹配度、偿债安全性，识别流动性与债务风险；
5. 测算自由现金流规模与可持续性，评估公司分红、回购、偿债、再投资的能力，判断股东长期回报潜力；
6. 结合现金流组合特征，判断公司所处生命周期阶段，与第一部分行业生命周期分析交叉验证。

##### 分析联动规则

1. **与第一部分联动**：盈利现金比率、销售收现比率异常，需与2.1审计意见、2.2会计政策、2.3异常数据初筛的结果交叉验证，完成财报真实性校验的闭环；现金流组合定性需与1.2行业生命周期分析交叉验证；
2. **与本部分其他模块联动**：
   - 经营现金流回款/付现分析，需与模块1资产负债表的应收/应付、上下游话语权分析联动；
   - 资本开支、投资现金流分析，需与模块1资产负债表的固定资产、在建工程分析联动；
   - 筹资现金流、利息支出分析，需与模块1资产负债表的有息负债、偿债能力分析联动；
   - 盈利现金比率、销售收现比率，需与模块2利润表的营收、净利润、盈利质量分析联动；
   - 核心指标ROE、自由现金流、权益乘数，需与模块4杜邦分析体系联动；
3. **与后续模块联动**：现金流质量、自由现金流、偿债能力的分析结果，需纳入第五部分风险识别与敏感性分析、第六部分估值定价，作为DCF估值、股息贴现模型的核心参数。

### 模块4 杜邦分析体系

#### 模块目标

以\*\*净资产收益率（ROE）\*\*为核心，完成以下分析目标，形成三大财务报表分析的闭环收口：

1. 拆解ROE的三层驱动因素，量化判断公司盈利模式的本质（高净利率/高周转/高杠杆驱动）；
2. 通过近3-5年历史趋势分析，验证ROE驱动因素的稳定性与可持续性；
3. 通过细分赛道行业对标，判断公司ROE的相对竞争力、核心优势与短板；
4. 穿透底层指标，拆解净利率、总资产周转率的核心驱动，验证前序利润表、资产负债表的分析结论；
5. 通过管理用杜邦分析，区分经营损益与金融损益，精准判断公司核心经营资产的盈利能力，剔除杠杆与融资的干扰；
6. 识别ROE驱动中的潜在风险，为后续风险识别、估值定价提供核心依据。

*适配行业：煤化工、石油化工、煤炭、有色金属、钢铁、建材、航运、养殖等强周期行业*

#### 模块目标

基于周期行业「供需决定价格、价格决定盈利、产能决定周期、成本决定穿越周期能力」的核心逻辑，完成以下分析目标：

2. 拆解行业供需格局，识别供需错配的核心驱动、持续性与拐点信号；
3. 量化验证公司**成本护城河的深度与可持续性**，明确其在行业成本曲线中的位置；
4. 分析公司产能壁垒、投放节奏与行业周期的匹配度，判断逆周期布局能力；
5. 结合周期位置判断估值合理性，识别周期向上/向下的拐点信号；
6. 验证周期波动下公司现金流的安全性与抗风险能力。

## 第四部分 同业对标与护城河验证

### 第四部分全行业适配总规则

1. 本部分核心目标：通过**精准同业对标**量化公司相对竞争优势/劣势，交叉验证前序模块分析结论，最终完成公司护城河的真实性、深度与可持续性验证，同时为后续估值定价提供可比基准；
2. 可比公司筛选标准（强制执行）：
   - 数量：细分赛道内**3-5家核心可比公司**，形成头部梯度对标；
   - 匹配度：业务结构重合度≥60%，细分赛道完全一致，避免跨赛道无效对标；
   - 规模可比性：市值/产能/营收规模差异≤3倍，剔除极端多元化、ST、重大资产重组标的；
   - 数据有效性：上市时间≥3年，财务数据完整合规，审计意见为标准无保留；
3. 所有数据需明确标注来源，严格执行「数据校验规则」，异常值必须标注原因与影响；
4. 状态标注统一规范：✅ 已有 / ⚠️ 待提取 / 🔍 待搜索 / ⚙️ 待计算；
5. 全行业适配：核心对标维度、护城河验证重点必须与第三部分「分行业差异化分析框架」完全匹配，避免通用框架水土不服。

### 全行业适配调整提示

1. 强周期行业，**价格波动、成本上涨、供需格局**为核心敏感因素，极端压力测试必须包含价格跌至成本线场景；
2. 消费行业，**品牌力、渠道动销、食品安全**为核心敏感因素，极端压力测试必须包含品牌负面舆情场景；
3. 医药行业，**集采降价、研发失败、专利到期**为核心敏感因素，极端压力测试必须包含核心产品集采降价90%场景；
4. 金融行业，**资产质量、流动性、监管政策**为核心敏感因素，极端压力测试必须包含资产质量恶化+流动性危机场景；
5. 高端制造行业，**技术迭代、供应链断供、客户集中**为核心敏感因素，极端压力测试必须包含核心客户流失场景；
6. 房地产行业，**销售回款、土储减值、债务违约**为核心敏感因素，极端压力测试必须包含销售下滑30%+融资收紧场景。

## 第六部分 估值与价值判断

### 第六部分全行业适配总规则

1. **核心定位**：本部分为全框架投资决策收口模块，以前序财报真实性校验、财务分析、行业周期判断、同业对标、风险识别为核心输入，通过适配的估值方法完成公司内在价值测算，最终形成可落地的投资判断，与前序所有模块形成价值判断闭环。
2. **估值方法行业适配强制执行规则**（与第三部分分行业框架完全匹配）：
   | 行业类型 | 优先估值方法 | 辅助估值方法 | 严格禁用方法 | 核心逻辑 |
   |----------|--------------|--------------|--------------|----------|
   | 强周期行业（化工/煤炭/有色/钢铁/航运） | PB-LF、EV/EBITDA、周期调整PE（CAPE） | 股息贴现模型（DDM） | 永续增长DCF、静态PE | 周期波动大，盈利在周期顶部/底部失真，资产价值、息税折旧前盈利更稳定 |
   | 成长行业（高端制造/新能源/半导体/互联网） | PEG、PS、EV/EBITDA | 两阶段DCF | 静态PE、PB | 盈利处于高速增长期，短期利润不能反映长期价值，需匹配增速、市占率估值 |
   | 成熟行业（必选消费/公用事业/家电） | 静态PE、DDM、EV/EBITDA | PB、DCF | 高增速PEG | 盈利稳定、现金流可预测，盈利与分红能力是核心估值锚 |
   | 金融行业（银行/保险/证券/信托） | PB-LF、PE（券商）、DDM、内含价值（PEV，保险） | 股息率 | EV/EBITDA、DCF | 经营杠杆高、息税折旧前盈利无法反映资产质量，净资产、内含价值是核心估值基础 |
   | 医药行业（创新药/器械/CXO） | PEG、DCF（管线现金流折现）、PS | PE（成熟仿制药）、EV/EBITDA | 静态PB | 创新药管线价值需通过现金流折现体现，成长型标的需匹配研发进度与增速 |
   | 房地产行业 | PB-NAV（净资产重估）、PS | PE（存量物业） | DCF、EV/EBITDA | 项目制经营，盈利结转波动大，净资产重估、销售规模是核心估值锚 |
3. **估值参数校验规则（强制执行）**：
   - 所有估值参数必须来自前序分析结论、可比公司公告、行业协会、头部券商深度研报，禁止无依据的主观假设；
   - 可比公司必须与第四部分同业对标标的完全一致，保证估值对标与竞争分析的统一性；
   - 周期行业估值必须结合第三部分周期位置判断，周期顶部慎用高PE倍数，周期底部慎用低PB倍数；
   - 高风险标的必须计提风险折价，折价比例与第五部分风险等级严格匹配：中风险折价10%-20%，高风险折价30%-50%；
   - 估值结果异常值（与当前价格偏离±100%以上）必须标注原因，重新校验参数合理性。
4. **状态标注统一规范**：✅ 已有 / ⚠️ 待提取 / 🔍 待搜索 / ⚙️ 待计算。
5. **估值前提红线**：前序2.1审计意见为非标准无保留、财报真实性校验存在高风险的标的，仅可做资产清算估值，禁止开展持续经营假设下的投资估值。
