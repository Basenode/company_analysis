# 网络信息搜索规范（WebSearch Guidelines）

> 本文件为大模型执行网络信息搜索时的约束规范，确保搜索结果的准确性、时效性和可靠性。

---

## 搜索原则

### 1. 核心原则
- **准确性优先**：优先选择权威来源，避免使用自媒体、博客等非官方信息
- **时效性要求**：优先选择最新发布的信息，标注发布时间
- **多源验证**：关键信息需要至少2个独立来源交叉验证
- **来源标注**：所有搜索结果必须标注来源URL和发布时间

### 2. 年份动态处理规则（重要）

**⚠️ 禁止硬编码年份**：

| 占位符 | 替换规则 | 示例 |
|--------|----------|------|
| `{最新年份}` | 替换为**当前实际年份** | 2026年执行时填 `2026` |
| `{最新季度}` | 替换为**当前实际季度** | 2026年3月执行时填 `2026Q1` |
| `{最新月份}` | 替换为**当前实际月份** | 2026年3月执行时填 `2026年3月` |

**执行规则**：
- ❌ 禁止：搜索关键词写死 `2024` 或 `2025`
- ✅ 正确：使用当前实际年份（从系统环境 `<env>` 中的 `Today's date` 获取）
- 年份必须动态获取，不得在模板或示例中硬编码

### 3. 禁止行为
- ❌ 不得使用单一来源的关键信息
- ❌ 不得使用发布时间超过2年的信息（除非是历史数据）
- ❌ 不得使用无法验证真实性的信息
- ❌ 不得使用付费内容或需要注册才能查看的内容
- ❌ 不得使用竞争对手的营销材料作为客观信息

---

### 3. 权威来源分级规则

**来源等级判定标准**：

| 来源等级 | 类型 | 示例 | 权重 | 适用场景 |
|----------|------|------|------|----------|
| **S级** | 政府/监管机构/交易所公告 | 证监会、工信部、上交所/深交所公告、国家统计局 | 100% | 政策解读、行业分类、监管处罚 |
| **A级** | 公司年报/招股书/行业协会数据 | 公司年报、招股书、中汽协/中钢协统计数据 | 90% | 财务数据、业务描述、行业规模 |
| **B级** | 头部券商研报/权威财经媒体 | 中信/中金研报、财新网/第一财经报道 | 80% | 行业分析、竞争格局、投资逻辑 |
| **C级** | 第三方数据机构/普通财经媒体 | 艾瑞咨询、东方财富网报道 | 60% | 补充数据、市场情绪 |

**头部券商分级**：
- **S级券商**：中信证券、中金公司
- **A级券商**：国泰君安、华泰证券、招商证券、海通证券
- **B级券商**：区域性券商、中小券商

---

### 4. 交叉验证规则

**验证要求细化**：

「至少2个来源确认」需满足以下条件之一：
- **条件1**：1个S/A级来源 + 1个任意级来源
- **条件2**：2个B级及以上来源

**交叉验证冲突处理规则**：

| 冲突类型 | 处理方式 |
|----------|----------|
| **同等级来源冲突** | 补充至少1个更高等级来源验证 |
| **不同等级来源冲突** | 优先采用更高等级来源 |
| **无法补充验证** | 标注「信息冲突」，说明各来源核心差异，不强行下结论 |

**冲突处理示例**：
```
⚠️ 数据交叉验证差异：
- 来源A（S级-国家统计局）：{数值A}
- 来源B（B级-券商研报）：{数值B}
- 差异说明：{具体差异描述}
- 采用值：{数值}，采用原因：优先采用S级官方统计数据
```

## 搜索内容分类

### 1. 公司底层定位信息（第一优先级）

#### 1.1 核心主营业务与商业模式

> **模块关联说明**：本模块是公司底层定位的核心，其结论将作为1.2行业属性判断、1.3竞争格局分析、1.4分析目标锚定的基础依据。

**搜索关键词模板**：

**通用关键词**：
```
"{公司名} {细分赛道} 同行 商业模式对比"
"{公司名} {核心产品} 单位成本 行业排名"
"{细分赛道} 应收应付 预付预收 行业平均水平"
"{细分赛道} 监管政策 对头部企业影响"
```

**分行业适配关键词**：
```
# ToC消费行业
"{公司名} {核心产品} 复购率 客单价 渠道占比 {最新年份}"

# ToB工业行业
"{公司名} {核心产品} 大客户占比 账期 集采中标 {最新年份}"

# 金融行业
"{公司名} 净息差 不良率 资本充足率 同行对比 {最新季度}"

# 周期行业
"{公司名} 产能利用率 开工率 库存周期 {最新年份}"
```

**必查内容**：

**核心必查（无替代）**：
- 公司主营业务构成（产品/服务类型、营收占比）
- 盈利模式（如何赚钱、收入来源）
- 客户类型（2B/2C/2G属性）
- 资产属性（轻重资产、资本密集度）

**补充必查（可替代）**：
- **三类话语权验证**：
  - 上游话语权：应付账款/预付账款行业对标
  - 下游话语权：应收账款/合同负债行业对标
  - 政府话语权：行业政策对头部企业的影响
- **成本结构**：固定成本/变动成本占比、核心成本构成
- **现金流特征**：经营/投资/筹资现金流净额占比、现金流与净利润匹配度
- **全球化属性**：海外营收/利润占比、海外业务商业模式

**数据来源优先级**：

| 优先级 | 来源类型 | 示例 |
|--------|----------|------|
| 1 | 公司官方 | 公司官网、年报主营业务章节、招股书业务模式章节 |
| 2 | 监管机构 | 交易所公告、业务变更公告 |
| 3 | 行业专属 | 消费行业：天猫/京东品类财报；工业行业：产业链上下游企业公告 |
| 4 | 权威财经媒体 | 财新网、第一财经行业分析文章 |

**验证要求**：
- ✅ 至少2个来源确认主营业务构成（需满足「1个S/A级 + 1个任意级」或「2个B级及以上」）
- ✅ 营收占比数据需与年报财务数据一致
- ✅ 商业模式描述需与实际业务匹配
- ✅ 话语权指标需有行业对标数据

**数据获取失败提示（差异化方案）**：
```
⚠️ 数据获取失败：{具体数据项名称}
- 尝试来源：{data_pack_market.md §X / pdf_sections.json [章节名]}
- 失败原因：{数据不存在/章节未找到/格式异常}
- 替代方案：
  ▶ 若为营收占比：优先搜索{公司名}最新业绩说明会纪要 + 同花顺财经营收拆解
  ▶ 若为成本结构：优先搜索{公司名}年报管理层讨论与分析章节 + 券商研报成本拆解
  ▶ 若为话语权指标：优先搜索{细分赛道}行业研究报告 + 同行财务数据对比
```

#### 1.4 分析目标锚定

> **模块关联说明**：本模块需基于1.1-1.3的分析结论，锚定市场共识与核心分歧，明确后续分析的重点验证方向；其结论将直接指导第二部分财务分析的侧重点。

**搜索关键词模板**：

```
"{公司名} 机构持仓 社保 外资 基金持股比例 {最新季度}"

"{公司名} 核心投资逻辑 券商研报 {最新年份}"
"{公司名} 投资风险 市场分歧 券商研报 {最新年份}"

```

**市场情绪/舆情维度关键词**：
```
"{公司名} 舆情热度 负面舆情 整改 {最新月份}"
"{公司名} 融资融券 余额 变动 {最新季度}"
```

- 机构投资者持股比例

### 1. 新兴行业/跨界公司适配

**适用场景**：

**适配规则**：

| 场景 | 处理方式 |
|------|----------|
