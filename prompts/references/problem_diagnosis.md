# 问题诊断框架 — 深度分析规则

> 本文件为 Phase 3 执行器的参考文件。当财报质量评估或因子分析中发现异常信号时，Agent 3 按需加载本文件执行问题诊断。
> 所有数据均从 data_pack_market.md 和 data_pack_report.md 已有字段读取，不可随意猜测。

---

## 触发条件

当以下任一条件满足时，执行问题诊断：

| 触发信号 | 来源 | 诊断重点 |
|:---------|:-----|:---------|
| 营收增速 < 0% | 模块〇-B (1) | 业绩下滑诊断 |
| 净利润增速 < 营收增速 且 净利润增速 < 0% | 模块〇-B (1) | 业绩下滑诊断 |
| 净现比 < 0.7 | 模块〇-B (3) | 现金流问题根源 |
| 应收占营收比 > 30% 或 YoY增幅 > 50% | 模块〇-B (3) | 现金流问题根源 |
| 毛利率 YoY下降 > 5pct | 模块〇-B (2) | 成本压力分析 |
| ROE YoY下降 > 5pct | 模块〇-B (2) | 综合诊断 |
| 财务风险评级 = 高风险 | 模块〇-B (4) | 综合诊断 |
| 因子1A触发否决 | 因子1A | 针对性诊断 |

## 一、业绩下滑诊断

### 1.1 营收下降的幅度和趋势

**数据来源**：data_pack_market.md §3（利润表）、§12（关键财务指标）

| 分析维度 | 计算公式 | 数据来源 | 判断标准 |
|:---------|:---------|:---------|:---------|
| 营收降幅 | (上期营收 - 本期营收) / 上期营收 | §3 营业收入 | <5%轻微 / 5-15%中等 / >15%严重 |
| 连续下滑年数 | 统计营收连续下降的年数 | §3 营业收入多年数据 | 1年偶发 / 2年趋势 / ≥3年结构性 |
| 季度趋势 | 最近4个季度营收环比 | §3 营业收入（若有季度数据） | 企稳回升 / 持续下滑 / 波动剧烈 |

**输出**：
```
营收降幅：[X]%，程度：[轻微/中等/严重]
下滑持续时间：[X]年，性质：[偶发/趋势/结构性]
季度趋势：[企稳回升/持续下滑/波动剧烈]
```

### 1.2 行业性问题 vs 个体问题

**数据来源**：data_pack_market.md §8（行业与竞争）、§13（风险预警）

| 判断维度 | 数据来源 | 行业性问题信号 | 个体问题信号 |
|:---------|:---------|:---------------|:-------------|
| 同行表现 | §8 行业趋势 + WebSearch | 行业整体下滑 | 行业增长但公司下滑 |
| 产品价格 | §8 行业趋势 | 行业产品价格普跌 | 公司定价能力下降 |
| 需求变化 | §8 行业趋势 | 行业需求萎缩 | 公司市场份额丢失 |
| 政策影响 | §8 行业趋势 + §13 | 行业政策收紧 | 公司合规问题 |

**判断逻辑**：
```
行业性问题：同行表现一致下滑 + 产品价格普跌 + 需求萎缩
个体问题：行业增长但公司下滑 + 市场份额丢失 + 公司特有风险
混合因素：行业下滑 + 公司下滑幅度远超同行
```

**输出**：
```
问题性质：[行业性问题/个体问题/混合因素]
判断依据：[≤3点]
同行对比：[公司表现 vs 行业均值]
```

### 1.3 季度间波动分析

**数据来源**：data_pack_market.md §3（利润表，若有季度数据）

| 分析维度 | 计算方法 | 判断标准 |
|:---------|:---------|:---------|
| 季度环比波动 | 各季度营收环比变化率 | 波动>20%为剧烈 |
| 季节性规律 | 与历史同期对比 | 是否偏离季节性规律 |
| 下滑加速/减速 | 连续季度降幅变化 | 加速下滑/企稳/反弹 |

**输出**：
```
季度波动幅度：[剧烈/中等/平稳]
季节性偏离：[是/否]，偏离程度：[X]%
下滑趋势：[加速下滑/企稳中/已反弹]
```

### 1.4 与历史同期对比

**数据来源**：data_pack_market.md §3（利润表多年数据）

| 对比维度 | 计算方法 | 判断标准 |
|:---------|:---------|:---------|
| 历史周期对比 | 本期营收 vs 历史周期底部营收 | 是否跌破历史底部 |
| 历史均值对比 | 本期营收 vs 5年均值 | 偏离程度 |
| 恢复时间估算 | 基于历史恢复速度 | 预计恢复所需时间 |

**输出**：
```
历史位置：[高于均值/低于均值/接近历史底部/跌破历史底部]
偏离均值：[X]%
历史恢复参考：[上次类似下滑恢复耗时X年]
```

## 二、现金流问题根源

### 2.1 应收账款激增原因

**数据来源**：data_pack_market.md §3（利润表）、§4（资产负债表）、data_pack_report.md P3（应收账款账龄）

| 分析维度 | 计算公式 | 数据来源 | 判断标准 |
|:---------|:---------|:---------|:---------|
| 应收增幅 | (本期应收 - 上期应收) / 上期应收 | §4 应收账款 | >营收增速为异常 |
| 应收周转天数变化 | 本期周转天数 - 上期周转天数 | 365 × §4 应收账款 ÷ §3 营业收入 | 增加>15天为恶化 |
| 账龄结构 | 1年以内 / 1-2年 / 2年以上占比 | data_pack_report.md P3 | 长期账龄占比上升为风险 |
| 坏账准备率 | 坏账准备 / 应收账款原值 | data_pack_report.md P3 | >10%高风险 |

**应收激增原因判断**：

| 原因类型 | 判断依据 | 风险程度 |
|:---------|:---------|:---------|
| **信用政策放宽** | 应收增幅 > 营收增幅 且 无新大客户 | 中风险 |
| **大客户账期延长** | 前五大客户应收占比上升 | 中风险 |
| **回款能力下降** | 周转天数大幅增加 + 坏账率上升 | 高风险 |
| **收入确认激进** | 应收激增但现金流未改善 | 高风险 |
| **行业账期延长** | 行业整体账期延长（需WebSearch验证） | 中风险 |

**输出**：
```
应收增幅：[X]%，vs 营收增幅：[Y]%，判断：[正常/异常]
周转天数变化：[+X]天，趋势：[恶化/稳定/改善]
账龄结构：1年内[X]%，1-2年[Y]%，2年以上[Z]%
激增原因：[信用政策放宽/大客户账期延长/回款能力下降/收入确认激进/行业因素]
风险程度：[低/中/高]
```

### 2.2 行业话语权问题

**数据来源**：data_pack_market.md §4（资产负债表）、§8（行业与竞争）

| 分析维度 | 计算公式 | 数据来源 | 判断标准 |
|:---------|:---------|:---------|:---------|
| 应付账款周转天数 | 365 × 应付账款 / 营业成本 | 365 × §4 应付账款 ÷ §3 营业成本 | 上升=话语权增强 |
| 预收账款/合同负债 | 预收账款 / 营业收入 | §4 预收账款或合同负债 ÷ §3 营业收入 | 高=话语权强 |
| 上下游议价能力 | (应付-应收) / 总资产 | (§4 应付账款 - §4 应收账款) ÷ §4 总资产 | 正值=占用他人资金 |

**行业话语权判断**：
```
强势：预收占比高 + 应付周转天数长 + 应收周转天数短
弱势：预收占比低 + 应付周转天数短 + 应收周转天数长
中性：介于两者之间
```

**输出**：
```
行业话语权：[强势/中性/弱势]
应付周转天数：[X]天，应收周转天数：[Y]天
预收/合同负债占营收：[X]%
上下游资金占用：[占用他人X百万 / 被占用X百万]
```

### 2.3 外汇结算延迟

**数据来源**：data_pack_market.md §3（利润表）、§4（资产负债表）

**触发条件**：海外收入占比 > 20% 或 有明显外汇相关披露

| 分析维度 | 数据来源 | 判断标准 |
|:---------|:---------|:---------|
| 海外收入占比 | §9 主营业务构成 或 data_pack_report.md | >20%需关注外汇风险 |
| 汇率波动影响 | §13 风险预警 或 年报附注 | 汇兑损益占净利润比例 |
| 外币应收占比 | data_pack_report.md P3（若有） | 外币应收占比 |

**输出**：
```
外汇风险敞口：[高/中/低/不适用]
海外收入占比：[X]%
汇兑损益影响：[X]百万元，占净利润：[Y]%
外汇结算延迟风险：[有/无]
```

### 2.4 存货积压

**数据来源**：data_pack_market.md §3（利润表）、§4（资产负债表）

| 分析维度 | 计算公式 | 数据来源 | 判断标准 |
|:---------|:---------|:---------|:---------|
| 存货增幅 | (本期存货 - 上期存货) / 上期存货 | §4 存货 | >营收增速为积压信号 |
| 存货周转天数变化 | 本期周转天数 - 上期周转天数 | 365 × §4 存货 ÷ §3 营业成本 | 增加>15天为恶化 |
| 存货跌价准备率 | 跌价准备 / 存货原值 | data_pack_report.md P2（若有） | >5%为风险信号 |

**存货积压原因判断**：

| 原因类型 | 判断依据 | 风险程度 |
|:---------|:---------|:---------|
| **需求下滑** | 存货增 + 营收降 + 行业需求萎缩 | 高风险 |
| **备货过度** | 存货增 + 营收稳/增 + 周转恶化 | 中风险 |
| **产品滞销** | 存货增 + 毛利率降 + 跌价准备增 | 高风险 |
| **季节性备货** | 存货增但符合季节性规律 | 低风险 |

**输出**：
```
存货增幅：[X]%，vs 营收增幅：[Y]%，判断：[正常/积压]
周转天数变化：[+X]天，趋势：[恶化/稳定/改善]
积压原因：[需求下滑/备货过度/产品滞销/季节性备货]
跌价风险：[低/中/高]
```
