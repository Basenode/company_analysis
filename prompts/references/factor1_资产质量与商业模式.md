## 二、因子1A：五分钟快筛 `[LLM-REQUIRED]`

**指令**：仅使用公开信息，逐条判断以下6项。使用**事件分级矩阵**对负面事件进行分级，而非简单一票否决。

> **注意**：即使触发Hard Veto，报告仍需包含完整的商业模式分析，帮助用户理解否决原因和公司基本面。

### 事件分级矩阵（新增）

| 分级 | 代码 | 触发条件 | 投资结论 | 后续核验要求 |
|------|------|----------|----------|--------------|
| **Hard Veto** | HV | 监管立案/司法判决/经营实质影响已确认 | ❌ 不建议投资 | 需持续跟踪监管进展、处罚结果、整改情况 |
| **Soft Veto** | SV | 负面信号存在，但原因未明/影响待观察 | ⚠️ 观察期（6-12个月） | 必须检查：监管是否跟进、公司公告说明、管理层稳定性、质押减持变化 |
| **Observation** | OB | 潜在风险信号，需密切监控 | ⚠️ 降低仓位或提高安全边际要求 | 纳入持续监控清单，定期核验 |
| **Info Only** | IO | 信息性事件，无实质影响 | ✅ 正常分析 | 记录备查 |

### 六快筛检查项（带分级）

| 序号 | 检查项 | Hard Veto | Soft Veto | Observation | 信息来源 |
|------|--------|-----------|-----------|-------------|----------|
| 1 | 审计意见异常 | 保留/否定/无法表示意见 | 强调事项段 | 标准无保留 | §7.1 审计意见 |
| 2 | 频繁更换审计师 | 5年≥2次+大所换小所 | 5年≥2次 | 5年1次 | §7.2 管理层信息 |
| 3 | 财务造假或重大违规 | 证监会立案/刑事判决 | 行政处罚/交易所谴责 | 历史记录已整改 | §7.2 管理层信息 |
| 4 | 看不懂商业模式 | 完全无法理解 | 部分业务模糊 | 清晰可理解 | 业务描述 |
| 5 | 商业模式未验证 | 无周期验证+重大转型中 | 转型初期<2年 | 转型>2年有进展 | 发展历程 |
| 6 | 控股股东负面信号 | 质押>70%/刑事调查/经营影响确认 | 质押50-70%/诉讼未决/原因未明事件 | 质押<50%/历史已解决 | §16 股权质押 + §7.2 |

**事件后续核验步骤**（凡触发SV/OB/HV必须执行）：

1. **监管动作检查**：证监会/交易所是否有立案、处罚、问询函

4. **量化缓释项**：质押率变化、回购动作、分红稳定性

  事件时间线：
    [日期] [事件] [来源]
  后续进展核验：
    - 监管动作：[有/无] [详情]

### 模块〇：数据校验与口径锚定 `[LLM-REQUIRED]`

> 本模块在深度定性分析前，快速完成数据校验和口径决策。
> 后续模块和因子直接从 data_pack_market.md 原表读数，不再搬运。

**行业前置（必须）**：
- 先从 data_pack_market.md §1 读取“适配行业类型/建议执行因子/禁用估值方法/核心驱动因子”。
- 若行业类型为“金融”（银行/保险/券商）：本框架的因子2-4不直接适用；模块〇只做“口径说明+缺口清单+需人工确认项”，并在报告中明确后续因子将关闭或替代。

**指令**：快速扫描 data_pack_market.md §3-§5、§12、§13.1，完成以下3项判断。

**(1) 异常扫描**

- 扫描 §3/§4/§5 各指标，标注任何 YoY 变动 >30% 或利润率变动 >5pct 的项
- 每项给出简要原因（非经常性损益/周期波动/并购/会计政策变更等）
- 上限3条，超过3条仅列最重要的

**(2) 利润口径锚定**

- 判断后续因子2/3应使用哪个利润口径：
  GAAP归母净利润 / 扣非归母净利润 / 主营经营利润
- 给出选择理由（如"其他收入超过经营利润30%，应使用扣非口径"）
- 明确标注：锚定口径 = [X]，对应 data_pack §3 行项 = [行项名]

> **交叉校验**：将 §3 自行计算的比率与 §12（Tushare 财务指标）中的预计算值（如 grossprofit_margin、netprofit_margin、roe 等）进行交叉验证。若偏差 > 2pct，标注差异原因（口径/四舍五入/调整项）。

**(3) 现金口径决策**

- 判断是否采用广义现金口径（含定期存款/理财产品）
- 判断依据：公司是否在流动性披露中将定期存款/理财与现金并列
- **受限资金剔除**：检查 data_pack_report.md P2「受限资产」中是否有「货币资金」受限项
  - 若有受限货币资金 → 真实可用现金 = §4 货币资金 - P2受限货币资金
  - 若无受限货币资金 → 直接使用 §4 货币资金
- 明确标注：使用 [狭义/广义] 现金口径，data_pack §4 行项 = [行项名列表]，受限资金 = [值]

> **现金口径说明**：
> - **狭义现金** = 现金及等价物 + 短期投资（传统口径）
> - **广义现金** = 狭义现金 + 持有至到期定期存款/理财产品（若公司将其视为流动性储备）
> - **可用现金** = 现金口径 - 受限货币资金（信用证保证金、冻结资金、矿山生态保证金等）

**中期数据标注**（若有中期列）：
最新中期 = [2025Q3]，年化系数 = 4/3
后续因子读取当期值时，应同时列出 FY 值和年化值

输出：

### 模块〇-B：财报质量评估 `[COMPUTABLE]` + `[LLM-REQUIRED]`

> 本模块对财务报表质量进行系统性评估，所有数据均从 data_pack_market.md 已有字段读取，不可随意猜测或估算。

#### (1) 业绩成长性分析

**数据来源**：data_pack_market.md §3（利润表）、§12（关键财务指标）

| 分析维度 | 计算公式 | 数据来源 | 判断标准 |
|:---------|:---------|:---------|:---------|
| 营收增速(同比) | (本期营收-上期营收)/上期营收 | §3 营业收入 YoY | >20%高增长 / 10-20%稳健 / <10%低速 / <0%下滑 |
| 营收增速(环比) | (本期营收-上期营收)/上期营收 | §3 营业收入 QoQ | 环比趋势是否与同比一致 |
| 净利润增速(同比) | (本期净利润-上期净利润)/上期净利润 | §3 归母净利润 YoY | 与营收增速对比，判断利润质量 |
| 扣非vs非扣非 | 扣非净利润 / 归母净利润 | §3 扣非净利润 ÷ §3 归母净利润 | >90%高质量 / 70-90%一般 / <70%低质量 |

**合同负债前瞻指标**：
```
合同负债变动 = 本期合同负债 - 上期合同负债（§4 合同负债）
合同负债/营收 = 合同负债 / 营业收入（§4 合同负债 ÷ §3 营业收入）
判断：合同负债上升 + 占比提升 → 未来收入确认潜力大
```

**业绩可持续性判断**：
- 增长来源：内生增长 vs 外延并购（检查 §4 商誉变动）
- 增长驱动：量增 vs 价增 vs 毛利改善（结合 §9 主营业务构成）
- 周期性影响：若为强周期行业，需剔除周期因素

输出：
```
营收增速：[X]% YoY，[Y]% QoQ，判断：[高增长/稳健/低速/下滑]
净利润增速：[X]% YoY，与营收增速对比：[同步/领先/滞后]
扣非净利润占比：[X]%，利润质量：[高/中/低]
合同负债变动：[增加/减少] [X] 百万，占营收 [Y]%，前瞻信号：[正面/中性/负面]
业绩可持续性：[强/中/弱]，理由：[≤2点]
```

#### (2) 盈利能力拆解

**数据来源**：data_pack_market.md §3（利润表）、§12（关键财务指标）

**ROE杜邦拆解**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利润/营收) × (营收/总资产) × (总资产/净资产)

数据来源：
  净利率 = §12 净利率 或 §3 归母净利润 ÷ §3 营业收入
  资产周转率 = §12 总资产周转率 或 §3 营业收入 ÷ §4 总资产
  权益乘数 = 1 / (1 - §12 资产负债率) 或 §4 总资产 ÷ §4 股东权益合计
```

| 指标 | 数据来源 | 本期值 | 上期值 | 变动 | 判断 |
|:-----|:---------|:-------|:-------|:-----|:-----|
| 毛利率 | §12 毛利率 | {值}% | {值}% | {Δ}pct | 趋势判断 |
| 净利率 | §12 净利率 | {值}% | {值}% | {Δ}pct | 与同行对比 |
| ROE | §12 ROE | {值}% | {值}% | {Δ}pct | 拆解来源 |
| 资产周转率 | §12 总资产周转率 | {值} | {值} | {Δ} | 效率变化 |
| 权益乘数 | 计算 | {值} | {值} | {Δ} | 杠杆变化 |

**ROE来源质量判断**：
```
高效率驱动：资产周转率 > 同行均值 且 杠杆率 < 同行均值 → 优质
高杠杆驱动：杠杆率 > 同行均值 且 效率 < 同行均值 → 风险
均衡型：效率与杠杆均接近同行均值 → 中性
```

输出：
```
毛利率：[X]%，趋势：[上升/稳定/下降]，原因：[≤1点]
净利率：[X]%，与同行对比：[高于/持平/低于]
ROE拆解：净利率[X]% × 周转率[Y] × 权益乘数[Z] = [W]%
ROE来源：[效率驱动/杠杆驱动/均衡型]，质量：[优质/中性/需警惕]
```

#### (3) 现金流健康度

**数据来源**：data_pack_market.md §5（现金流量表）、§3（利润表）

| 指标 | 计算公式 | 数据来源 | 判断标准 |
|:-----|:---------|:---------|:---------|
| 收现比 | 销售收现 / 营业收入 | §5 销售商品收到现金 ÷ §3 营业收入 | >1.1优秀 / 0.8-1.1健康 / <0.8风险 |
| 净现比 | 经营现金流 / 净利润 | §5 经营活动现金流 ÷ §3 归母净利润 | >1.2优秀 / 0.7-1.2健康 / <0.7风险 |
| 应收账款周转天数 | 365 × 应收账款 / 营业收入 | 365 × §4 应收账款 ÷ §3 营业收入 | 与同行对比 |
| 应收占营收比 | 应收账款 / 营业收入 | §4 应收账款 ÷ §3 营业收入 | <15%优秀 / 15-30%正常 / >30%风险 |
| 存货周转天数 | 365 × 存货 / 营业成本 | 365 × §4 存货 ÷ §3 营业成本 | 与同行对比，趋势判断 |

**现金流质量综合判断**：
```
收现比 + 净现比 双优 → 现金流质量高
收现比优 + 净现比差 → 可能有资本支出过大/应收增加
收现比差 + 净现比优 → 可能有预收增加/应付增加
双差 → 现金流质量低，需深入分析
```

输出：
```
收现比：[X]，判断：[优秀/健康/风险]
净现比：[X]，判断：[优秀/健康/风险]
应收周转天数：[X]天，占营收比：[Y]%，判断：[优秀/正常/风险]
存货周转天数：[X]天，趋势：[加快/稳定/放缓]
现金流综合质量：[高/中/低]
```

#### (4) 财务风险评估

**数据来源**：data_pack_market.md §4（资产负债表）、§12（关键财务指标）、data_pack_report.md P2（受限资产）

| 风险维度 | 指标 | 计算公式 | 数据来源 | 警戒线 |
|:---------|:-----|:---------|:---------|:-------|
| 短期偿债 | 现金覆盖短债 | (可用现金) / 短期有息负债 | (§4 货币资金 - P2受限货币资金) ÷ (§4 短期借款+一年内到期非流动负债) | >1安全 / 0.5-1关注 / <0.5风险 |
| 流动性 | 流动比率 | 流动资产 / 流动负债 | §12 流动比率 | >2优秀 / 1-2正常 / <1风险 |
| 杠杆水平 | 资产负债率 | 总负债 / 总资产 | §12 资产负债率 | <40%低 / 40-60%正常 / >60%偏高 |
| 应收风险 | 应收坏账准备率 | 坏账准备 / 应收账款原值 | data_pack_report.md P3（若有） | >10%高风险 |
| 商誉风险 | 商誉占净资产比 | 商誉 / 归母净资产 | §4 商誉 ÷ §4 归母权益 | >30%高风险 / 10-30%关注 / <10%低风险 |
| 其他应付款 | 其他应付款异常 | 其他应付款 / 总资产 | §4 其他应付款 ÷ §4 总资产 | 占比突增需关注 |

> **短期偿债计算说明**：
> - 若 P2 无受限货币资金 → 直接用 §4 货币资金
> - 若 P2 有受限货币资金 → 可用现金 = §4 货币资金 - P2受限货币资金
> - 受限货币资金包括：信用证保证金、冻结资金、矿山生态保证金等无法自由支配的资金

**财务风险综合评分**：
```
低风险：所有指标均在安全区间
中风险：1-2个指标触及关注线
高风险：任一指标触及警戒线 或 多个指标同时触及关注线
```

输出：
```
可用现金：[X]百万元（§4 货币资金 [Y]百万元 - P2受限资金 [Z]百万元）
现金覆盖短债：[X]倍，判断：[安全/关注/风险]
流动比率：[X]，判断：[优秀/正常/风险]
资产负债率：[X]%，杠杆水平：[低/正常/偏高]
商誉占净资产：[X]%，商誉风险：[低/中/高]
其他应付款占比：[X]%，异常：[有/无]
财务风险综合评级：[低风险/中风险/高风险]
```

### 模块三：竞争格局与护城河

输出：`[优质 / 中性 / 负面]`
